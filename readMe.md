# URL Shortner

A simple and efficient URL Shortener web application that converts long URLs into short, shareable links. This project provides a user-friendly API and interface to generate shortened URLs and redirect users to their original links.

## About

This project is designed to make long and complex URLs shorter and easy to share with others. Shortened URLs can be used in messaging, social platforms, email, and other mediums where character count or readability matters.

## Features

-  Generate short URLs from long URLs  
-  Redirect users from short links to original URLs  
-  custom alias for short url
-  qr code for easy shareability
-  link expiration 
-  link management(read,update,delete)
